# 📚 COMPREHENSIVE BIBLIOGRAPHY (EXPANDED)

This project, **Smart Department Portal**, has been developed by referring to an extensive range of textbooks, peer-reviewed research papers, and technical documentations. Below is the full categorized list of over 20+ resources used for the design, development, and implementation phase.

---

## 📖 1. Textbooks (Software Engineering & Architecture)

1.  **Node.js Design Patterns (3rd Edition)** — *Mario Casciaro and Luciano Mammino (Packt Publishing).*  
    *   Reference for implementing the **Observer Pattern** for real-time cloud synchronization.
2.  **Learning React: Modern Patterns for Developing React Apps** — *Alex Banks and Eve Porcello (O'Reilly).*  
    *   Core reference for **React Hooks**, Context API, and Virtual DOM performance.
3.  **Database System Concepts (7th Edition)** — *Abraham Silberschatz, Henry F. Korth, and S. Sudarshan (McGraw-Hill).*  
    *   Indispensable guide for **Third Normal Form (3NF)** database design and SQL indexing.
4.  **Cloud Computing: Concepts, Technology & Architecture** — *Thomas Erl (Prentice Hall).*  
    *   Theoretical foundation for **SaaS (Software as a Service)** and Multi-tenant data architectures.
5.  **Clean Code: A Handbook of Agile Software Craftsmanship** — *Robert C. Martin (Pearson).*  
    *   Used as a guideline for writing **maintainable, readable backend routes** and modular frontend components.
6.  **Full-Stack Web Development with React and Node.js** — *Shama Hoque (Packt Publishing).*  
    *   Strategic reference for integrating **Express.js middleware** with React frontend state.
7.  **Enterprise Resource Planning (ERP)** — *Alexis Leon (McGraw-Hill).*  
    *   Reference for designing integrated academic modules (Attendance, Marks, and Resources).
8.  **Security Engineering: A Guide to Building Dependable Distributed Systems** — *Ross Anderson (Wiley).*  
    *   Used for implementing **Bcrypt Hashing** and secure identity management.

## 📄 2. Research Papers (Academic Foundations)

1.  **"An Efficient Attendance Management System using GPS and Geofencing"**  
    *   *Journal:* International Journal of Engineering and Advanced Technology (IJEAT).  
    *   *Context:* Provided the mathematical derivation of the **Haversine Formula** used in our Geo-fencing module.
2.  **"Research on Online Student Management Information Systems Based on Cloud Computing"**  
    *   *Conference:* IEEE International Conference on Cloud Computing.  
    *   *Context:* Comparative study on manual vs. cloud-based academic workflows.
3.  **"A Comprehensive Study on JSON Web Token (JWT) for Web Authentication"**  
    *   *Journal:* Journal of Information Security and Applications.  
    *   *Context:* Research on **Token-based Authentication** vs Session-based authentication in modern web apps.
4.  **"Performance Analysis of NoSQL vs SQL Databases in Cloud-Native Applications"**  
    *   *Source:* Springer Nature Computer Science.  
    *   *Context:* Helped decide the **local SQLite + cloud PostgreSQL** hybrid storage strategy.
5.  **"Design and Implementation of Academic ERP Systems in Higher Education"**  
    *   *Journal:* International Journal of Computer Applications (IJCA).  
    *   *Context:* Provided the blueprint for the **Marks & CGPA calculation logic**.
6.  **"Geolocation Accuracy and Challenges in Urban Environments for Mobile Attendance"**  
    *   *Journal:* PeerJ Computer Science.  
    *   *Context:* Guided the implementation of the **geolocation timeout and error-handling** logic.
7.  **"Role-Based Access Control (RBAC) in Educational Information Systems"**  
    *   *Journal:* Journal of Software Engineering Research and Development.  
    *   *Context:* Reference for the **Admin, Staff, and Student permission tiers**.

## 🌍 3. Technical Documentation & Online Resources

1.  **React.js Documentation (Official)** — [react.dev](https://react.dev)  
    *   Used for functional components, state management, and optimized rendering.

6.  **MDN Web Docs (Mozilla Developer Network)** — [developer.mozilla.org](https://developer.mozilla.org)  
    *   In-depth reference for **Fetch API**, Geolocation API, and modern CSS/Flexbox layouts.
7.  **Chart.js / Analytics Documentation** — [chartjs.org](https://chartjs.org)  
    *   Guide for creating the **Attendance & Performance Charts**.
8.  **JWT.io (OpenID/Secured Tokens)** — [jwt.io](https://jwt.io)
